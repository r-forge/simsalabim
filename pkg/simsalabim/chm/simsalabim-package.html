simsalabim-package(simsalabim)R Documentation

A collection of methods for time series analysis and signal detection.

Description

This package implements Singular System Analysis and Phase Synchronization Indices.

Details

The functions bundled in this package focus on two objectives: The application of Singular Spectrum Analysis (SSA) and the computation of phase synchronization measures. The working horses of this package are the functions decompSSA and phaSyn. For both applications some statistical testing procedures are provided. SSA is implemented in a standard form and in a variant able to handle missing data points decompSSAM. Currently the entire package is implemented in R which leads to some performance issues.
